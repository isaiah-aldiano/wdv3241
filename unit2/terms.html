version control software - version control is a form of content management that aids programmers in maintaining concurrent changes made on a single project
add - moves changes made on a local repo to a staging area to be commited
commit - creates a snapshot of changes in staging area to be merged with the local repo
push - merges or creates a request to merge a programmers local changes to the remote repo
pull - applies any changes made to the remote repo to a users local verison
clone - points at a remote repo and copies all of the files in that repo to a local directory Storing doc in wdv341 folder - add this html file to my local wdv341 folder
